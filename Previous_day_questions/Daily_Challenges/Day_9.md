## 1. Length of Last Word (LeetCode #58)
A problem that focuses on string traversal by determining the length of the last word in a given string. Given a string s consisting of words and spaces, return the length of the last word in the string.

This problem is commonly asked in interviews and helps build concepts like:
- String traversal
- Word extraction
- Handling whitespace
- String processing

which are important for text-processing problems.

**Your task:** Find the length of the last word using an optimized approach.

### Examples

**Input:**
```
s = "Hello World"
```
**Output:**
```
5
```

---

**Input:**
```
s = " fly me to the moon "
```
**Output:**
```
4
```

---

**Input:**
```
s = "luffy is still joyboy"
```
**Output:**
```
6
```

## 2. Remove Spaces from String
A problem that focuses on string manipulation by removing all space characters from a given string. Given a string s, remove all spaces from the string and return the modified string.

This problem helps build concepts like:
- String traversal
- Character filtering
- In-place modification
- Basic string manipulation

which are important for solving many string-based programming problems.

**Your task:** Remove all spaces from the given string using an optimized approach.

### Examples

**Input:**
```
s = "C od ing"
```
**Output:**
```
"Coding"
```

---

**Input:**
```
s = "abc d "
```
**Output:**
```
"abcd"
```
