## 1. Rotate List (LeetCode #61)
A problem that teaches linked list traversal and pointer manipulation techniques by rotating a linked list to the right by k positions. Given the head of a linked list and an integer k, rotate the list to the right by k places and return the updated head.

This problem is commonly asked in interviews and helps build concepts like:
- Linked list traversal
- Circular linking
- Pointer manipulation

which are important for efficient linked-list problems.

**Your task:** Rotate the linked list to the right by k positions using an optimized approach.

### Examples

**Input:**
```
head = [1,2,3,4,5], k = 2
```
**Output:**
```
[4,5,1,2,3]
```

---

**Input:**
```
head = [0,1,2], k = 4
```
**Output:**
```
[2,0,1]
```

## 2. Remove Nth Node From End of Linked List
A problem that teaches linked list traversal and two-pointer techniques by deleting a node from a specific position counted from the end of the list. Given the head of a linked list and an integer n, remove the nth node from the end of the linked list and return the updated head.

This problem is commonly asked in interviews and helps build concepts like:
- Linked list traversal
- Node deletion

which are important for efficient linked-list operations.

**Your task:** Remove the nth node from the end of the linked list using an optimized approach.

### Examples

**Input:**
```
head = [1,2,3,4,5], n = 2
```
**Output:**
```
[1,2,3,5]
```

---

**Input:**
```
head = [1], n = 1
```
**Output:**
```
[]
```

---

**Input:**
```
head = [1,2], n = 1
```
**Output:**
```
[1]
```
