## 1. Merge Two Sorted Lists (LeetCode #21)
A problem that teaches linked list merging and pointer traversal techniques by combining two sorted linked lists into one sorted list. Given the heads of two sorted linked lists, merge them into a single sorted linked list and return its head.

This problem helps build concepts like:
- Linked list traversal
- Pointer manipulation
- Merging techniques

These concepts are useful for efficient merging problems.

**Your task:** Merge the two sorted linked lists using an optimized approach.

### Examples

**Input:**
```
list1 = [1,2,4], list2 = [1,3,4]
```
**Output:**
```
[1,1,2,3,4,4]
```

---

**Input:**
```
list1 = [], list2 = []
```
**Output:**
```
[]
```

---

**Input:**
```
list1 = [], list2 = [0]
```
**Output:**
```
[0]
```

---

## 2. Reverse a Singly Linked List
A problem that teaches linked list traversal and pointer manipulation techniques by reversing the direction of a singly linked list. Given the head of a singly linked list, reverse the linked list and return the head of the reversed list.

This problem helps build concepts like:
- Pointer manipulation
- Linked list traversal
- Iterative reversal

These concepts are useful for efficient linked-list operations.

**Your task:** Reverse the given singly linked list using an optimized approach.

### Examples

**Input:**
```
1 -> 2 -> 3 -> 4 -> 5
```
**Output:**
```
5 -> 4 -> 3 -> 2 -> 1
```

---

**Input:**
```
1 -> 2
```
**Output:**
```
2 -> 1
```

---

**Input:**
```
1
```
**Output:**
```
1
```

---
